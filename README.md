# react-media-form

A simple project containing only 1 component to test out storybook + chromatic
https://www.chromatic.com/component?appId=5f2ad8210a5a7e0022c8ec14&name=Welcome

## Instruction
1. Install dependencies
   ```sh
   yarn install
   ```

2. Run storybook
   ```sh
   yarn storybook
   ```
